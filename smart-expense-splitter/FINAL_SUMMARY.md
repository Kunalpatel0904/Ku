## ❓ FAQ

**Q: Do I need PostgreSQL for local development?**  
A: No! SQLite is set up by default. PostgreSQL is optional for production.

**Q: Can I deploy without a database?**  
A: No, you need a database. Supabase & Railway offer free PostgreSQL tiers.

**Q: How do I create demo data?**  
A: Just follow the demo script - create a group, add members, add expenses. The app auto-generates the data.

**Q: What if the demo is too long?**  
A: Edit it down. The requirement is **< 5 minutes**. Loom has built-in editing.

**Q: Can I deploy to Netlify instead of Vercel?**  
A: Netlify has limitations with Next.js. Vercel is recommended.
